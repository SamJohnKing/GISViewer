This is a cross-platform key-value relationship based spatial database designed by Java
 
Library Dependence:
Lwjgl 2.8.5 jar and dll native files
eclipse.swt win64 version and IE browser 
Oracle jre 7, 8 or Openjdk 7, 8

未完心愿：对于PointDB,LineDB,PolygonDB目前是数组加字符串存储，更好的实现方法是四分树+字母树的存储方式，四分树解决空间坐标系统问题，字母树解决键值对问题，四分树要支持读写多线程锁
